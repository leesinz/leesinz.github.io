java基础-反射

基础篇

概念

Java反射就是在运行状态中，对于任意一个类，都能够知道这个类的所有属性和方法；对于任意一个对象，都能够调用它的任意方法和属性；并且能改变它的属性。而这也是Java被视为动态语言的一个关键性质。并且java反序列化也离不开java的反射机制。

反射相关的类和方法

java.lang.Class：类对象

java.lang.reflect.Constructor：类的构造器对象

java.lang.reflect.Field：类的属性对象

java.lang.reflect.Method：类的方法对象

forName：获取类的方法

newInstance：实例化类对象的方法

getMethod：获取函数的方法

invoke：执行函数的方法
